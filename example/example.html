One Size Images

To show ho to use the Justified Gallery with only one type of images. In this case it use only the '_t' images that is very small, and this creates a low quality gallery. This can be done modifying the sizeSuffixes setting and setting all the prefixes to the same. If you have images without prefixes, you can set all the prefixes to the empty string 'sizeSuffixes' : { 'lt100':'', 'lt240':'', 'lt320':'', 'lt500':'', 'lt640':'', 'lt1024':''} 
Settings: no lightbox, rowHeight = 250, no fixed height, justify last row
