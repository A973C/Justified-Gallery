Justified Gallery Examples

Simple example

Here we have a div, with the id myExample0, that has a list of links that point to some images (big images), where each link has a thumbnail. Calling it, these thumbnail can be changed, and bigger or smaller thumbnails can be loaded, to have always an high quality. Remember that the default value for the rowHeight is 120, and the thumbnails with the longest side of 240px (i.e. '_m' suffix) is a good choice to have speed and low bandwidth. 
<div id="myExample0">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_m.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_m.jpg" />
	</a>
	...
</div>

$("#myExample0").justifiedGallery();

Small gallery, no justify last row, margins

This is an example where we use smaller images. Look that the thumbnails are small images (i.e. use the '_t' suffix, that corresponds to images with a longest side less than 100px), because is very lickely that, with a rowHeight = 110, the used images will be quite small. In this way the plugin probably will not load other images, with a huge performance improvement.
Furthermore, note that we don't justify the last row, and that we are using a margin = 5px. 
<div id="myExample1">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$("#myExample1").justifiedGallery({
	'rowHeight':70,
	'justifyLastRow':false, 
	'margins':3
});

Lightbox

In this example we use the Colorbox lightbox. To do this we define the onComplete callback function, and in this function we call Callbox. Note that we also are using the rel = 'gallery2' setting, that set to 'gallery2' all the links rel attribute value. In this way Colorbox can create the gallery, showing them as a group. 
<div id="myExample2">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$("#myExample2").justifiedGallery({ 
	'justifyLastRow':false, 
	'rowHeight':70, 
	'rel':'gallery2', //replace the rel of all the links with the value 'gallery2'
	'margins':1,
	'onComplete': function(gal) { //callback function, the gal parameter is the gallery that has been built.
		$(gal).find("a").colorbox({
			maxWidth : "80%",
			maxHeight : "80%",
			opacity : 0.8,
			transition : "elastic",
			current : ""
		});		
	}
});

Fixed Height, no caption, target _blank

In this example you can see how the fixed height option works. Note that the big images is cutted, because they have a maximum height that they need to respect. Also, we disable the captions, and then we set the images to be opened in a new page (i.e. each link target is set to '_blank'). 
<div id="myExample3">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_m.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_m.jpg" />
	</a>
	...
</div>

$("#myExample3").justifiedGallery({
	'rowHeight' : 200, 
	'fixedHeight' : true,
	'captions' : false,
	'target' : '_blank',
	'margins' : 2
});

One Size Images

To show how to use the Justified Gallery with only one type of images. In this case it use only the '_t' images that is very small, and this creates a low quality gallery. This can be done modifying the sizeRangeSuffixes setting and setting all the prefixes to the same. This is the case where you have only one type of thumbnail (e.g. "myimage_t.jpg") and the big original image (e.g. "myimage_b.jpg"). 
<div id="myExample6" >
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$("#myExample6").justifiedGallery({
	'sizeRangeSuffixes' : {
		'lt100':'_t', 
		'lt240':'_t', 
		'lt320':'_t', 
		'lt500':'_t', 
		'lt640':'_t', 
		'lt1024':'_t'
	},
	'rowHeight':250
});
